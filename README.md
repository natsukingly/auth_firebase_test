## Description

## Install

```
git clone https://github.com/xojan0120/auth_firebase_test.git
cd auth_firebase_test
bin/bundle install
bin/rails db:migrate
bin/rails server
```

## Firebase Configuration
* app/assets/javascripts/firebase.js (Initialize Firebase Section)
* config/firebase.yml
